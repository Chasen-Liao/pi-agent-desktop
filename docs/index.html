Features & Specs

为高效代码智能体工作流而生

剔除繁杂，专注于最关键的上下文、分支探索、以及原生 Electron 的无缝接入。

01 // WORKSPACE TREE

会话工作区

按项目工作目录（CWD）自动组织历史会话树，使多项目并发开发时的上下文浏览、切换和恢复更加直观、扁平。

02 // DOUBLE BRANCHING

双轨分支探索

支持物理 Fork（创建独立 session.jsonl 文件）与会话内分支切换（通过 navigate_tree 回退到历史节点继续对话），完美契合 AI 多样化试错。

03 // CODEGRAPH INTEGRATION

CodeGraph 深度集成

对于索引过的项目，内置 CodeGraph MCP 工具可以直接进行高效的跨文件符号关系、定义及引用搜索，无需在大量无关文件之间反复切换。

04 // TABBED WORKBENCH

标签页式工作台

内置轻量级代码浏览器与 Tab 切换机制，使智能体的输出、工具链的调用与实际的源文件保持在同一屏中，上下文零断层。

Technical Stack

轻量级双向融合架构

Pi Agent Desktop 使用 Next.js / React 搭建交互层，Electron 提供桌面化外壳，通过 SSE (Server-Sent Events) 与 Pi 智能体核心保持高频实时通信。

APP // NEXT.JS FRONTEND

React & Next.js UI

复用现代化 Web UI，基于 TailwindCSS 和 Vanilla CSS 进行主题与响应式精细调整。通过自定义 SSE client 监听智能体实时响应。

SHELL // ELECTRON STANDALONE

Electron 原生托盘

独立窗口运行，支持快速唤醒、最小化到系统托盘，并集成自动更新检查模块。生产环境下通过 Node.js 进程级托管 Next.js 服务端。

Setup & Deployment

三步快速运行工作台

你可以通过克隆源码或下载最新版发行程序，在本地快速接入你的 Pi coding agent。

pi-setup.sh
1. 安装依赖软件包
npm install copy
2. 启动 Web 开发服务器 (默认端口: 30141)
npm run dev copy
3. 以 Electron 桌面模式启动独立客户端
npm run dev:electron copy
